MSetPoint

MSetPoint version 0.9

MSetPoint, which stands for MIDAS Set Point, is an extension of the MIDAS data acquisition system designed for beam line control through the EPICS channel access system. It provides real-time values for various beamline elements and offers a user-friendly interface to modify these values. Additionally, MSetPoint supports saving and loading these values from files.

For further information on MIDAS, please visit the MIDAS home page. For further information about MSetPoint, visit the online documentation on bitbucket.
